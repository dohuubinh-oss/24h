6   Cài đặt plugin:
Chạy lệnh sau trong terminal:
bash
npm install -D @tailwindcss/typography
# hoặc pnpm add -D @tailwindcss/typography
Cấu hình trong file CSS chính (Tailwind 4):
Mở file CSS toàn cục của bạn (ví dụ: app/globals.css) và import plugin:
css
@import "tailwindcss";
@plugin "@tailwindcss/typography";
